EcoValuator

  • A python plugin for QGIS
  • Funded by National Fish & Wildlife Foundation

A simple means of estimating the dollar value of ecosystem services in a study area

Ecosystem Serivices

The benefits that people obtain from the natural environment

  • Provisioning Services
    • provides food, fresh water, fuel...
  • Regulating Services
    • influences climate, water, pollination
  • Supporting Services
    • soil formation, nutrient cycling
  • Cultural Services
    • educational, aesthetic, recreation, cultural value

Current Landcover Rasters

Each dataset categorizes the earth into about 20 landcover types

NLCD: National Land Cover Dataset

NALCMS: North American Land Change Monitoring System

Step 1: Estimate Ecosystem Service Values for Study Region

  • Clips input land cover raster to the extent of user's study area
  • Calculates how much area each type of land cover is present in study area
  • Multiplies those areas by each of the associated per-hectare ecosystem services values

Benefit Transfer Method

  • Provides an accessible way of estimating the value of ecosystem service flows based on the values estimated in another similar setting.

  • Referenced studies performed in other, similar ecosystems to extrapolate the value of each ecosystem service

Step 2: Map the Value of Individual Ecosystem Services

Uses step 1 output and creates a new raster for which the value is represented per-pixel for the user-selected ecosystem service
